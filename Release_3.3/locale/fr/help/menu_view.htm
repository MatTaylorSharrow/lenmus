Menu vue

Comme aller là-bas

Le menu vue c'est un item du menu de la fenêtre principale.

Qu'est ce qu'il fait

Le menu vue contient diverses ordres se rapportant aux options de visualisation, comme la barre d'outils de visualisation, barre de status, règles, outiles, etc...

Ordres

Bouton Nom d'ordre Raccourci Description
  Barre d'outils  Montre ou cache la barre d'outils principale.
  barre de status  Montre ou cahe la barre de status au fond de la fenêtre principale.
