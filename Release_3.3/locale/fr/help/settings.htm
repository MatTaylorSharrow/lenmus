Préférences et configuration de sons

Dialogue de préférences

Le dialogue de préférences est valide à travers Options/Préférences dans le menu principal ou cliquant le bouton sur la barre d'outils principale.

Vous pouvez mettre diverses options qui affectent globalement à LenMus. Il y a trois groups de configuration.

Langage

Vous autorise à choisir le langage des eBooks et l'interface d'utilisateur. Le langage choisi será applique pour la prochaine fois que vous demarrez LenMus. Attention: seulement seront montrés les eBooks dont le langage soit le même que celui de l'inteface d'utilisateur. Cela implique que le contenu réel du logiciel LenMus peux varier, dependant du langage choisi.

Barre d'outils

Vous pouvez choisir diverses options pour la barre d'outils, soit utilisant seulement icones, sans text, montrant les icones et le text en diverses positions ou changeant la taille des icones de la barre d'outils.

Autres

Autres options pas inclues dans les groups de configuration d'avant. Courammant, la seule configuration dans ce group c'est l'option pour essayer automatiquement l'actualisation de logiciels, se brancheant à l'Internet au demarrage du logiciel.

Configuration de sons

Le génie du Midi vous permet de configurer l'interface Midi, requise pour syntétiser les sons. Le génie vous guide pour accomplir les diverses pas requises.

C'est valide à travers Sound/Run Midi wizard.
