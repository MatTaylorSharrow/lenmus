Menu sons

Comme aller là-bas

Le menu de sons c'est un item de la fenêtre principale:

Qu'est ce qu'il fait

le menu de sons contient des ordres diverses se rapportant au son et son configuration.

Ordres

Bouton Nom d'ordre Raccourci Description
Jouer  Commence pour jouer la partition courante ou réprends le jeu dès le point où elle a été pausé.
Arrêt  Arrête le jeu de la partition courante.
Pause  Pause le jeu de la partition courante. Cique sur "Jouer" pour reprendre le jeu.
  Génie tourne-Midi  Déclenche une séquence de dialogue pour mettre à point l'interface de hardware MIDI, pour configurer la carte de son pour jouer des partitions et les cliques du métronome.
  Essais de son  Essaie la configuration du son MIDI courant pour produire une échelle de notes.
  Muet  Quand quelque problème arrive à l'ordinateur, il est possible que le hardware du Midi reste accroché, produisant un son continu. Cette ordre arrête immediatement tous les sons dans le hardware Midi.
