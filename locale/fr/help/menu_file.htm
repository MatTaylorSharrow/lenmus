Menu fichier

Comme aller là-bas

Le menu fichier c'est un item du menu de la fenêtre principale:

Qu'est ce qu'il fait

Coke la majorité des logiciels d'ordinateur, le fichier de menu de LenMus contient diverses ordres pour ouvrir, fermer, sauver et imprimer partitions et eBooks. Il contient aussi des ordres pour importer partitions en autres formats, habituellement en MusicXML. Finalement, le menu fichier conetient l'ordre Quitter pour sortir du logiciel.

Ordres

Bouton Nom d'ordre Raccourci Description
NouveauCtrl+N Créer une nouvelle partition vide.
OuvrirCtrl+O Ouvrir une partition LenMus existante (.ldp).
Livres ouverts  Ouvre eBooks gérant et montre la page d'introduction.
  Importer  Ouvre une partition MusicXML existante.
SauverCtrl+S Sauver la partition courante.
  Sauver comme...Ctrl+Shift+S Sauve la partition courante avec un nom different. Cela vous permet de choisir un nouveau nom pour la partition. L'ancient nom n'est pas éffacé donc il reste enregistré sur le disque.
  FermerCtrl+W ferme la fenêtre courante, vous démandant si vous voulez sauver les changements.
Imprimer...Ctrl+P Imprime la partition courante ou la page de l'eBook montré.
  Configuration d'imprimante...  Configure comme LenMus va imprimer la partition ouverte ou la page de l'eBook montrée quand-t-on utilise l'ordre imprimer et quelle imprimante à utiliser.
  Pre-vue d'impressionCtrl+Shift+P Ouvre une fenêtre pour montrer comme le partition ouverte ou la page de l'eBook montrée vont être imprimés quand-t-on utilise l'ordre imprimer.
  QuitterCtrl+Q ferme toutes les fenêtres et quityte LenMus, vous démandant d'abord si vous voulez sauver quelque change pas sauvé.
