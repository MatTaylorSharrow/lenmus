Gerant d'eBooks

Quand le boutton  est appuyé ou quand vous selecctionnez la ordre 'Livres ouverts' dans le 'fichier' menu principal, le gérant de la fenêtre du eBook est visible. Cette fenêtre est subdivisée en deux fenêtres:

1. Fenêtre de navigation

A gauche il y en a l'arbre de navigation qui montre les livres valides. Cliquant sur quelque item de cet arbre de navigation, le sujet d'exercice choisi será montré sur la fenêtre à droite (La fenêtre de documents). Vous pouvez cacher ou montrer cette fenêtre de navigation appuyant le bouton .

Sur le panneau de navigation vous trouverez des diverses options de recherche. Aussi, vous pouvez stocker des signets de vos livres et pages préférées. Au bout, utilisez les boutons suivants:

Bouton Description
Ajouter la page courante aux signets
Effacer la page courante des signets

2. Fenêtre de documents

La fenêtre de droite montre le contenu de l'exercice ou thème choisi. Vous pouvez cliquer sur le link dans cette page pour sauter ver d'autres thèmes.

la barre d'outils de eBooks

Pour gérer les livres, une nouvelle barre d'outils est ouverte, avec les boutons suivants:

 

Bouton Description
Cache ou montre la fenêtre de navigation.
Aller à la page anterieure du registre de navigation
Aller à la page prochaine du registre de navigation
Monter un niveau dans la hiérarchie du document.
Aller à la page anterieure du livre courant.
Aller à la page suivante du livre courant
Utilisez ce bouton pour imprimer la page montrée.
Ouvrez un dialogue pour selectionner la font et sa taille.
