Sound Menu

How to get there

The Sound menu is an item of main window menu:

What it does

The Sound menu contains various commands related to sound and sound settings.

Commands

Button Command name Shortcut Description
Play  Starts playing back the active music score or resumes playing from the point at which it was paused.
Stop  Stops playing back the current music score.
Pause  Pauses playing back the current music score. Click on 'Play' to resume playing.
  Run Midi wizard  Starts a sequence of dialogs to set up the MIDI hardware interface, to configure sound for score play back and metronome clicks.
  Test sounds  Tests current MIDI sound configuration by producing a scale of notes.
  All sounds off  When a problem occurs in the computer, it could happend that the Midi hardware get hooked, producing a continous sound. This command stops inmediately all sound production in the Midi hardware.
