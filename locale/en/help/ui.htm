User Interface Overview

See Menu and Toolbar for description of the main menu and toolbar icons.

LenMus Phonascus has two modes of operation:

  1. As a reader of interactive music books; and
  2. As a tool to create and edit music scores and to play them back. This second functionality (score editor / play back) is now under development and is disabled in current version.

Reading eMusicBooks

By pressing the     button, you can open or close this functionality. At program start, this button is pressed and the eBook manager window is open. See page eBook Manager for a description of the eBook manager window and its operation.

Creating / editing scores

This functionality is now under development and is disabled in current version. So it is not beign yet described in this help documentation.
