User Interface Overview

LenMus Phonascus has two modes of operation:

  1. As a reader of interactive music books; and
  2. As a tool to create and edit music scores and to play them back. This second functionality (score editor / play back) is now under development and is disabled in current version. Therefore, it is not described in this help document.

At program start, the the eBook manager window is open. See page eBook Manager for a description of the eBook manager window and its operation.

The main window menu and toolbar are described in page Menu and Toolbar.

For a description of user configurable preferences and sound settings see page Preferences and settings.
