Preferences and Sound Settings

Preferences Dialog

The Preferences dialog is available through Options/Preferences in the main menu, or by clicking the     button in the main toolbar.

You can set various options that affect LenMus globally. There are two tabs in the dialog box, one for each setting group.

Language

Allows you to choose the language to use for eBooks and user interface. The choosen language will be applied the next time the LenMus program is run. Important: Only eBooks whose language is the same than the user interface language will be shown. This implies than the real content of the LenMus program could vary, depending of the choosen language.

Toolbars

You can choose different options for the toolbars, such as using only icons without any text, displaying the icon text in different positions, or change the size of the toolbar icons.

Sound Settings

(Available through Sound/Run Midi wizard in the main menu.)

The Midi wizard allows to configure the MIDI interface, required to synthesize the sounds. The wizard will guide you to perform the different required steps.
