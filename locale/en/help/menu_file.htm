File Menu

How to get there

The File Menu is an item of main window menu:

What it does

As in many computer programs, LenMus's File menu contains various commands for opening, closing, saving, and printing scores and eBooks. It also contains commands for importing scores in other formats, currently only in MusicXML. Finally, the File menu contains the Quit command used to exit the program.

Commands

Button Command name Shortcut Description
NewCtrl+N Creates a new empty score.
OpenCtrl+O Opens an existing LenMus score (.ldp).
Open books  Opens eBooks manager and displays the introduction page.
  Import  Opens an existing MusicXML score.
SaveCtrl+S Saves the current score.
  Save as...Ctrl+Shift+S Saves the current score under different name. It lets you choose the new name for the score. The old one is not deleted so it also remains in the disk.
  CloseCtrl+W Closes the current window, asking you if you want to save changes.
Print...Ctrl+P Print out the currently open score or eBook displayed page.
  Print Setup...  Configure how LenMus will print out the currently open score or eBook displayed page when using the Print command, and what printer to use.
  Print PreviewCtrl+Shift+P Opens a window to show how the currently open score or eBook displayed page will be printed when using the Print command.
  QuitCtrl+Q Closes all windows and exits LenMus, prompting you to save any unsaved changes first.
