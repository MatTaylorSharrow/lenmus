La interfaz de usuario

LenMus Phonascus tiene dos modos de funcionamiento:

  1. Como un lector de libros interactivos sobre música; y
  2. Como un editor para crear, modificar y ejecutar (reproducción sonora) partituras. Esta funcionalidad de editar y reproducir partituras está actualmente en desarrollo. Por ello, las funcionalidades relacionadas están inhabilitadas en esta versión del programa y no se describen en esta documentación de ayuda.

Cuando se inicia el programa la ventana del gestor de libros aparece abierta. Consule la página El gestor de libros (eBooks) para una descripción del esta ventana, sus controles y su funcionamiento.

El menú de la ventana principal y la barra de herramientas aparecen descritos en la página El menú y la barra de herramientas.

Para una descripción de las distintas opciones configurables por el usuario y la forma de ajustar los sonidos consulte la página Preferencias y configuración del sonido.
