eBooks manager

When the     button is pressed or when you select the 'Open books' command in the 'File' main menu, the eBook manager window is visible. This window is subdivided in two windows:

1. Navigation window

On the left there is a navigation tree displaying the available books. By clicking on any item of this navigation tree the choosen subject or exercise will be displayed in the right window (the document window). You can hide or show this navigation window by pressing the     button.

In the navigation panel you will find different search options. Also, you can store bookmarks to your favorite books and pages. To this end use the following buttons:

Button Description
Add current page to bookmarks
Remove current page from bookmarks

2. Document window

The window on the rigth shows the content of the choosen theme or exercise. You can click on the links in that page to jump to other themes.

The eBooks toolbar

To manage the books a new toolbar is opened, with the following buttons:

 

Button Description
Hide or show the navigation window.
Go to previous page in navigation log
Go to next page in navigation log
Go one level up in document hierarchy.
Go back to previous page of current eBook.
Go to next page of current eBook
Use this button to print the displayed page.
Opens a dialog to select the fonts and font sizes to use.
