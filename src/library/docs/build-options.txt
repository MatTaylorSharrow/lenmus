Preprocessor symbols

Defined by compilers:

Microsoft VS:
    WIN32;_DEBUG;_CONSOLE;_USRDLL

GCC:
    _WIN32


Target system
-------------

_WIN32, WIN32
    Building for Windows, even for 64bits
    
None:
    Building for Linux


Type of building
----------------

Define one of these:

LM_DYNAMIC_LIB
    For building the library as a DLL

LM_STATIC_LIB
    For building the library as a static library
    
None:
    For using library source files directly, included in another project, as if they were part of that project source tree.
