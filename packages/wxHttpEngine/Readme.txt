 wxHTTPEngine ReadMe
 -------------------
 
 Website: http://wxcode.sourceforge.net/components/wxhttpengine
 Author: Angelo Mandato
 Version: 2.0
 Description:
 wxHTTPEngine is a library of classes to easily allow your wxWidgets software to work with web sites.
