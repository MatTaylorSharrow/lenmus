LenMus Phonascus 3.0 beta 1 Release Notes

What's New in 3.0 beta 1

Release 3.0 beta 1 is mainly a bug fix release. Improvements include:

  • Counters for right/wrong answers and total marks included in all exercises.
  • Buttons for answers were missing in scales exercise and are now included.
  • Name of intervals now displayed in ear training exercises when requesting to show the solution.
  • More than 7 bug fixes.

 

Bug fixed details

Description
Too much space between notes in scales exercises; sometimes the scale did not fit in the screen.
The problem has been solved.
Memory leaks in scales exercise.
The problem was caused because old exercise score was not deleted.
Interval name not displayed in ear interval exercise.
The problem was caused by lmInterval due to strings initialized only in one constructor. Now, initializations have been moved to a private method and it is invoked from both constructors.
In occations, bad answers given for the exercises.
The problem was caused by lmPaper optimization code: if the new score pointer has the same address than the previous one, lmPaper did not re-draw the score and the old non-valid bitmaps got displayed.
Wrong answers in key signatures exercise, Spanish version.
They were caused by some bad translated strings .
Scores not rendered in Windows 98.
The problem disappeared when changing the logical units to one tenth of millimeter.
Occasional crashes after changing interval settings in earIntervalCtrol.
They were caused because the answer buttons and their names changed but the displayed buttons were not re-generated to match the new exercise constrains.
