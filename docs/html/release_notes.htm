LenMus Phonascus 3.0 Release Notes

What's New in 3.0 (since 3.0 beta 1)

Release 3.0 include new features and improvements and some bug fixes:

  • Sound feedback (wrong/right answer sound) added to all exercises.
  • Automatically checks if a new version of the program is available.
  • A new menu item to access to LenMus website.
  • About box improved to facilitate user accessibility to information.
  • Several bug fixes (see details).

 

Bug fixed details

Description
Clefs reading exercise does not display any score.
The bug was caused by the fix to memory leaks problem (3/may); the fix code was, inadvertely, placed in the wrong place. Now fixed.
Button colours in keyboard for answers in build intervals exercises not reset.
As 'build interval' exercise was added later, existing code to reset keyboard did not take into account the later added keyboard. Now fixed.
In occasions, bad answers given for the exercises.
The problem was caused by rendering optimization code: if by chance the new score pointer has the same address than the previous one, it was not re-drawn and the previous score non-valid bitmaps got displayed. I tried to fix this bug (2-may-06) but the fix did not take into account all cases. Now I hope this problem is definitely fixed.
In first run installed languages are not recognized.
It was caused by not properly setting the locale folder, as paths were instantiated after the language was chosen.
Keyboard for answers in build intervals exercises disabled after first answer given.
It was due to forgeting to re-enable the keyboard after generating a new exercise.
Primary and secondary beams too near.
Distance has been adjusted.
Invalid scores were sometimes generated in Music Reading Level 2 eBook, lessons 13 and 16.
The problem was caused by patterns with barlines marks in erroneous places. Patterns in eBook now fixed.
