LenMus Phonascus 3.5 Release Notes

What's new in 3.5

  • A new eMusicBook included: Music Reading level 1
  • Replaced MDI user interface by a notebook tabbed interface.
  • The lenmus program now includes the Dutch language translation.
  • Added support to include images on eMusicBooks texts.
  • Score parameters in eBooks: added tag to specify a fixed metronome rate to play the score, instead of using user specified value in metronome control.
  • Improvements in automatic music composer:
    • Modified so that in final measure it never adds a note longer that a quarter note if the composed piece only uses quarter notes. This was needed for Music Reading Level 1 eBook, for the initial lessons.
  • Improvements in the Score internal model to support more features:
    • Support to include graphical objects (arrows, lines, circles, etc.) on a score. For now only lines are implemented.
    • Implemented metronome marks notations.
    • Added support to set up the size of a measure by specifying barline position.
    • Added support for fermatas on notes and rests.
    • Added support to have global, inheritable options in all score elements. Implemented support for the following options:
      • Option 'Staff.DrawLeftBarline' to control left barline drawing.
      • Option 'StaffLines.Hide' to control staff lines drawing.
      • Option 'StaffLines.StopAtFinalBarline' to allow for staff lines going to the right paper margin instead of finishing at the final barline.
      • Option 'Staff.UpperLegerLines.Displacement' to shift the first upper leger line so that it is closer/further than normal spacing. The other leger lines spacing is not changed.
  • Improvements in LDP language to support the new features. New tags added:
    • Implemented simple notation 'fermata', and compound notation '(fermata [above | below])' on notes and rests.
    • tag 'metronome' to define metronome notations.
    • tag 'graphic line' to add lines on a score.
    • tag 'spacer' for better controlling staff space.
    • tag 'opt' to add global, inheritable options to all other tags.
    • Added support for strings enclosed in double simple quotes as an alternative to double quotes.
