LenMus Phonascus 3.0 beta 1 Release Notes

What's New in 3.0 beta 1

Release 3.0 beta 1 is mainly a bug fix release. Improvements include:

  • Counters for right/wrong answers and total marks included in all exercises.
  • Name of intervals now displayed in ear training exercises when requesting to show the solution.
  • More than 6 bug fixes.

 

Details

The following table details all changes an new features included in this release. Type of change is encoded as follows:

  • FIX - Fix a bug
  • NEW - Added a new functionality to the program
  • GUI - Change in the user interface (to improve ergonomy, to prepare for new functionalities, ...)
  • TEC - Internal technical change. Does not add new functionality neither changes the GUI. It serves for other purposes: to enable future functionalities, to improve coding, to improve performance, etc.

 

Type Description
FIX Memory leaks in scales exercise.
The problem was caused because old exercise score was not deleted.
FIX Interval name not displayed in ear interval exercise.
The problem was caused by lmInterval due to strings initialized only in one constructor. Now, initializations have been moved to a private method and it is invoked from both constructors.
FIX In occations, bad answers given for the exercises.
The problem was caused by lmPaper optimization code: if the new score pointer has the same address than the previous one, lmPaper did not re-draw the score and the old non-valid bitmaps got displayed.
FIX Wrong answers in key signatures exercise, Spanish version.
They were caused by some bad translated strings .
FIX Scores not rendered in Windows 98.
The problem disappeared when changing the logical units to one tenth of millimeter.
FIX Occasional crashes after changing interval settings in earIntervalCtrol.
They were caused because the answer buttons and their names changed but the displayed buttons were not re-generated to match the new exercise constrains. Links for 'play' and 'show solution' now enabled/disabled when appropiate.
