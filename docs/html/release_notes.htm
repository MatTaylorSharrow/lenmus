LenMus Phonascus 3.1 Release Notes

What's new in 3.1

Release 3.1 include new features and improvements and some bug fixes:

  • LDP language improvements to support more features.
  • Overlayered voices and cross-staff voices.
  • Score titles and text on the score.
  • Better renderization of chords.
  • Optional indentation of first system.
  • Score titles and text on the score.
  • Instrument name for first system.
  • Instrument abbreviation for other systems.
  • All types of tuples supported.
  • Play back controls added to main toolbar.
  • Several bug fixes (see details).
