LenMus Phonascus 3.1 Release Notes

What's new in 3.1

Release 3.1 include new features, improvements and some bug fixes:

  • First release oriented to support score edition:
    • LenMus scores imported from text files.
    • New toolbars for the score editor: play back, zoom.
    • Export scores as bmp or jpg images.
  • Toolbars are now draggable and dockable.
  • Several improvements in score layout. For instance:
    • Stem lengths now ajusted to follow usual music engraving rules.
    • Top and bottom time signature numbers now centered one respect the other one.
    • Space now added before clefs, at start of a system.
  • LDP language improvements to support more features.
  • Several bug fixes (see details).
  • Several technical improvements.
