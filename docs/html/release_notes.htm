LenMus Phonascus 3.0 Release Notes

What's New in 3.1

Release 3.1 include new features and improvements and some bug fixes:

  • Sound feedback (wrong/right answer sound) added to all exercises.
  • Automatically checks if a new version of the program is available.
  • Menu item to access to LenMus website.
  • Several bug fixes (see details).

 

Bug fixed details

Description
In first run installed languages are not recognized.
It was caused by not properly setting the locale folder as paths were instantiated before the language was chosen.
Keyboard for answers in build intervals exercises disabled after first answer given.
It was due to forgeting to re-enable the keyboard after generating a new exercise.
