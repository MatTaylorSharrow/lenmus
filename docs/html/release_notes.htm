LenMus Phonascus 3.3 Release Notes

What's new in 3.3

Release 3.3 include new features, technical improvements and some bug fixes:

  • New option to enable/disable right/wrong answer sounds in exercises (user request #1559450).
  • Screen renderization code modified to deal with fractions of a pixel and add anti-aliasing. This technique greately improves visual quality of displayed scores.
  • Several bug fixes (see details) and technical improvements.

 

Bug fixed details

Description
Occasionally, crash when displaying and exercise.
The problem was caused by score formatter, when score width is not enough for drawing just one bar. Now solved.
