LenMus Phonascus 3.3 Release Notes

What's new in 3.3

Release 3.3 include new features, improvements and some bug fixes:

  • Screen renderization code modified to deal with fractions of a pixel and add anti-aliasing. This technique greately improves display quality of scores.
  • Several bug fixes (see details) and technical improvements.

 

Bug fixed details

Description
Scale printing problems: the printed score is too big.
The problem was caused by incorrect computation of print and print preview scaling factors. Now solved.
