LenMus 5.2 installation in Windows NT/2000/XP/Vista/7

Just run the installer; it will guide you through the installation process.

LenMus 5.2 installation in GNU/Linux

Lenmus does not include makefiles. All makefiles are generated with CMake build system. CMake can generate different kinds of native build files for your system (e.g. Unix Makefiles, Eclipse CDT 4.0 project files, Visual Studio project files, and much more!).

Prerequisites

LenMus needs the wxWidgets libraries, the Lomse library and the following additional libraries: portmidi, UnitTest++, FreeType2, libpng, zlib, sqlite3, boost-thread and boost-date-time. Most of them are normally already installed in your system.

Installation

This is the procedure for installing in Ubuntu "precise" (12.04) or LinuxMint 13 "Maya". Please, adapt these instructions for your Linux distro (prerequisites might change). If you would like to contribute, please email me your instructions and patches. Thank you.

1. Install requirements:

sudo apt-get install cmake libboost-date-time-dev libboost-thread-dev
sudo apt-get install libportmidi-dev libunittest++-dev libsqlite3-0 libsqlite3-dev
sudo apt-get install libfreetype6-dev libpng++-dev zlib1g-dev

2. Install the Lomse library:

sudo add-apt-repository ppa:lomse/ppa
sudo apt-get update
sudo apt-get install liblomse

3. Download LenMus tarball and extract sources:

mkdir build-lenmus ; cd build-lenmus
wget -O lenmus_5.2.tar.gz "http://sourceforge.net/projects/lenmus/files/Sources/lenmus_5.2.tar.gz/download"
tar zxvf lenmus_5.2.tar.gz

4. Generate the makefile, build and install:

mkdir build ; cd build
cmake -G "Unix Makefiles" ../lenmus-5.2
make -j2   #change 2 by the number of processors in your machine
sudo make install
