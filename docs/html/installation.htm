LenMus 3.4 installation

Microsoft Windows LenMus 3.4 installation

Automatic installation

Just run the installer:

  • For Widows 98:    lenmus_3.4_setup.exe
  • For Windows NT / 2000 / XP:    lenmus_3.4u_setup.exe
It will ask you the language to use (for the installer and also as default LenMus language), the path to install LenMus and ask you about installing optional components.

Important: I don't know why but sometimes, just after the installation, Windows does not recognize the new installed font and instead of notes and musical symbols you get letters and other signs on the scores. This can be solved just by opening the 'windows/font' folder with MSExplorer. With this action, Windows 'recognizes' the new font. Now close MSExplorer and restart LenMus.

Manual installation

LenMus does not require special installation procedures. Just copy the files wherever you'd like (DO NOT ALTER names and/or structure). Then install the LenMus music font. That's all. No registry keys or other stuff. Follow this procedure:

  1. Create a folder to install LenMus and copy the files; follow the structure described in next section.
  2. Install the font lmbasic.ttf contained in folder /bin. For example:
    1. Open Windows Control Panel
    2. Open Fonts
    3. Choose "File/install font" and follow instructions
  3. Optionally, create any desired shortcuts on desktop, programs menu, etc.

LenMus 3.4 directories

After installation, in the chosen installation folder you will find some documentation, such as licence and installation procedure, and the following folders and content:

Directory Files Contents
bin lenmus.exeExecutable
lmbasic.ttf  True Type font with music symbols. If you install LenMus without using the setup program, YOU MUST INSTALL this font in the Windows/fonts directory.
res   Icons, bitmaps and related files.
xrc *.xrc XRC resources used by the program.
scores   Sample scores. There are two subdirectories. One, named 'MusicXML' for MusicXML scores; load them using File/Import. And another, named 'samples', for LDP (LenMus score language) scores; load them using File/Open.
books xx/*.lmb There is a subdirectory (xx) for each supported language. Subdirectories are named, following RFC 1766, with the two letter language code (.i.e. 'en' for English, 'es' for Spanish, etc.) or with the five chars language-culture code (i.e. 'en-EN' for USA English or 'es-ES' for the Modern Spanish).

Contains all eMusicBooks in compiled format (lmb stands for LenMus Book).
temp empty Empty folder. Used during LenMus execution. Can be changed with the 'options' menu
locale   Contains all language dependent files. There is a subdirectory for each supported language, also named following RFC 1766.
locale/xx *.moThe .mo message translation catalogues, except for English language (not needed).
help.htb The help file
