LenMus 3.0 installation

Release: 3.0 beta x

Microsoft Windows LenMus 3.0 installation

Automatic installation

Just run the installer (lenmus_3.0.bx_setup.exe). It will ask you for the path to install LenMus and ask you about installing or not optional components.

Manual installation

LenMus does not requiere special installation procedures. Just copy the files wherever you'd like (DO NOT ALTER names and/or structure). Then install the LenMus music font. That's all. No registry keys or other stuff. Follow this procedure:

  1. Create a folder to install LenMus and unzip lenmus_30bx.zip there.
  2. Install the font LeMusNot.ttf contained in folder /bin. For example:
    1. Open Windows Control Panel
    2. Open Fonts
    3. Choose "File/install font" and follow instructions
  3. Optionally, create any desired shortcuts on desktop, programs menu, etc.

LenMus 3.0 directories

After installation, in the chosen installation folder you will find some documentation, such as licence and installation procedure, and the following folders and content:

Directory Files Contents
bin lenmus.exeExecutable
LeMusNot.ttf  True Type font with music symbols. If you install LenMus without using the setup program, YOU MUST INSTALL this font in the Windows/fonts directory.
res   Icons, bitmaps and related files. There are two subdirectories: one for bitmaps and another for icons.
xrc *.xrc XRC resources used by the program.
scores   Sample scores. There are two subdirectories. One, named 'MusicXML' for MusicXML scores; load them using File/Import. And another, named 'test, for LDP (LenMus score language) scores; load them using File/Open.
temp empty Empty folder. Used during LenMus execution. Can be changed with the 'options' menu
locale   Contains all language dependent files. There is a subdirectory for each supported language. Subdirectories are named, following RFC 1766, with the two letter language code (.i.e. 'en' for English, 'es' for Spanish, etc.) or with the five chars language-culture code (i.e. 'en-EN' for USA English or 'es-ES' for the Modern Spanish).
locale/xx *.moThe .mo message translation catalogs, except for English language (not needed).
help.htb The help file
*.htb Other .htb files are eMusicBooks.
