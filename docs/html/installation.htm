Microsoft Windows installation

Automatic installation

Only modern Windows versions (NT/2000/XP/Vista) are supported. Just run the installer (lenmus_3.6_setup.exe).

It will ask you which language to use (for the installer and also as the default LenMus language), the path to install LenMus and ask you about installing optional components.

Important: I don't know why but sometimes, just after installation, Windows does not recognize the newly installed font and instead of notes and musical symbols you get letters and other signs on the scores. This can be solved just by opening the 'windows/font' folder with MSExplorer. With this action, Windows 'recognizes' the new font. Now close MSExplorer and restart LenMus.

Manual installation

LenMus does not require special installation procedures. Just copy the files wherever you'd like (DO NOT ALTER names and/or structure). Then install the LenMus music font. That's all. No registry keys or other stuff. Follow this procedure:

  1. Create a folder to install LenMus and copy the files; follow the structure described in next section.
  2. Install the font lmbasic.ttf contained in folder /bin. For example:
    1. Open the Windows Control Panel
    2. Open Fonts
    3. Choose "File/install font" and follow the instructions
  3. Optionally, create any desired shortcuts on your Desktop, Program Menu, etc.

LenMus 3.6 directories

You will find some documentation in the installation folder after installation, such as licence and installation procedure, and the following folders and content:


 c:\Program files (or any other location you would prefer)
 lenmus
 bin - Executable files.
 lenmus.exe - program executable.
 lenmus.ini - user options file.
 lmbasic.ttf - True Type font (note 2).
 logs - log files.
 xrc - XRC resources used by the program.
 res - Icons, bitmaps and related files.
 locale - Language dependent files.
 xx - Language xx (note 1).
 books - eMusicBooks.
 xx - Language xx (note 1).
 temp - Used during LenMus execution.
 scores - user scores
 MusicXML - MusicXML scores; load them using File/Import.
 samples - LenMus scores; load them using File/Open.
 docs - Documentation (license, installation guide, ...)

Notes

  1. There is a subdirectory (xx) for each supported language. Subdirectories are named, following RFC 1766, with the two letter language code (.i.e. 'en' for English, 'es' for Spanish, etc.) or with the five chars language-culture code (i.e. 'en-EN' for USA English or 'es-ES' for the Modern Spanish).

  2. Contains music symbols. It must be installed in Windows font folder.
